解锁额外的内容

NeTSEC
2022-01-07
Minecraft

在 1.16.5+ 中，您可以解锁 Tardis 的内部（控制台室）、外部和控制台单元。‎

‎为此，我们使用 ‎‎‎‎Schematics（原理图）来执行此操作。‎

图表‎

‎原理图是一个 “蓝图”，其中包含Tardis内饰，外部或控制台等信息。‎

‎它可以存储在音速起子中。‎

‎用于解锁各种内饰，外部和控制台。‎

解锁对象‎

‎从1.5开始，有两种主要获取原理图的方法‎

  • ‎将中子光谱仪与物品一起使用‎
  • ‎从特定结构中的 Ship 计算机中提取数据片段。‎

‎下面列出了可解锁对象的列表：‎

  • 可解锁内容‎

获取原理图‎

中子光谱仪
  1. ‎放下中子光谱仪‎
  2. 打开中子光谱仪的 GUI‎
  3. 将项目添加到 GUI 的左侧插槽中。如果添加了有效项目，Spectromer将继续从该项目中”提取”信息，并且蓝色进度条将缓慢增加。‎
  5. ‎等到蓝色进度条完成”提取”过程。左上角的绿灯将亮起。‎
  7. ‎如果将鼠标悬停在右上角的屏幕显示上，则可以看到提取的原理图。‎
  9. ‎将音速起子放入右下角插槽中，将原理图下载到螺丝刀中。此插槽有一个指示这一点的轮廓。‎
  11. ‎光谱仪现在将继续将原理图下载到声波螺丝刀中。两盏红灯将亮起，第二个进度条将缓慢增加。‎
    1. 在此期间，您无法将声波螺丝刀移出插槽，否则下载将被取消‎
    2. ‎在此步骤中不要破坏光谱仪，否则原理图将丢失‎
  12. 下载过程完成后，红色指示灯将关闭，并且只有一个指示灯将亮起绿色。
    1. 音速起子现在包含原理图‎
  14. 您可以通过按住左键在音速起子中查看下载的原理图。‎
Ship Computer
  1. Right click with a Sonic Screwdriver on a Ship Computer block inside a generated structure.
  2. If a computer contains a Schematic, a message similar to “Downloaded Data Fragments!” will be shown above the hotbar.
  3. You can view the downloaded Schematic in your Sonic Screwdriver by holding Left Control.

Installing Schematic into Tardis

  1. Enter the Tardis you wish to install the Schematic for
  2. Right click with the Sonic Screwdriver into the Sonic Port Control of the Console
  3. The Tardis will install any schematic stored inside the Sonic Screwdriver.
    1. It will not install the schematic if the Tardis already contains the interior/exterior/console.
